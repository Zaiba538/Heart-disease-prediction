# Heart-disease-prediction
# ❤️ Heart Disease Prediction using Machine Learning

## 📖 Overview

Cardiovascular diseases are among the leading causes of death worldwide. Early detection can significantly improve treatment outcomes and reduce health risks.

This project leverages Machine Learning techniques to predict the likelihood of heart disease based on patient medical data. The notebook covers the complete data science workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and performance comparison.

---

## 🎯 Problem Statement

The objective of this project is to develop a machine learning model capable of accurately predicting whether a patient is at risk of heart disease using clinical and demographic features.

---

## 📊 Dataset Description

The dataset contains various health-related attributes such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol Level
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate Achieved
- Exercise-Induced Angina
- ST Depression
- Slope of Peak Exercise ST Segment
- Number of Major Vessels
- Thalassemia
- Heart Disease Target Variable

---

## 🔄 Project Workflow

### 1️⃣ Data Collection
- Loaded the heart disease dataset.
- Inspected dataset structure and dimensions.

### 2️⃣ Data Understanding
- Checked data types.
- Generated statistical summaries.
- Identified missing values and duplicates.

### 3️⃣ Exploratory Data Analysis (EDA)
- Visualized feature distributions.
- Analyzed correlations between variables.
- Investigated relationships between patient attributes and heart disease outcomes.

### 4️⃣ Data Preprocessing
- Feature selection.
- Data cleaning.
- Feature scaling using StandardScaler.
- Train-test splitting.

### 5️⃣ Model Development
Implemented and evaluated multiple machine learning algorithms:

- Decision Tree Classifier
- Pruned Decision Tree
- Bagging Classifier
- Random Forest Classifier
- Gaussian Naive Bayes
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

### 6️⃣ Model Evaluation
Models were assessed using:

- Accuracy Score
- Confusion Matrix
- Classification Report
- Training vs Testing Performance Comparison
